Conozcamos las Plantas

Evaluación

Texto

Para la evaluación de esta unidad se utilizarán dos herramientas básicas:

  • Rúbrica Exploradores de plantas
  • Una Prueba escrita que se realizará durante la última sesión de la unidad, para comprobar los conocimientos adquiridos. Para ello se facilitarán tareas de repaso. 
  • Algunas de las fichas de repaso enviadas son autocorregibles y conllevarán una nota.
